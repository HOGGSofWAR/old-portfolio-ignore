Resident Evil 2 Remake Site

This website was made for fun, to experiment with animations which can be used in web design, especially with a focus on parallaxing. Many items respond to the mouse scrolling and mouse hover, such as the header, the images which slide in to the left, the fixed scrolling window, the blog post previews which come in from opposing sides and a fixed footer.

This project was coded in Atom, using: HTML, Jade, CSS, Sass and Jquery. The compiler used was Prepros as I do not have a Mac so could not use Codekit.
